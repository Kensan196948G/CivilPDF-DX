# アーキテクチャ概要

## システム構成

```
┌─────────────────────────────────────────────────┐
│                  CivilPDF-DX                     │
├─────────────────┬───────────────────────────────┤
│   GUI App       │      Management Console       │
│   (Windows exe) │      (WebUI)                  │
│                 │                                │
│  PDF閲覧・編集   │  ユーザー管理・権限管理         │
│  注釈・赤入れ    │  文書管理・ワークフロー         │
│  OCR・電子印鑑   │  監査ログ・ダッシュボード       │
│  オフライン作業   │  AI/OCR管理・外部連携          │
├─────────────────┴───────────────────────────────┤
│                  Backend API                     │
│              FastAPI + PostgreSQL                │
├─────────────────────────────────────────────────┤
│                Authentication                    │
│         Entra ID + HENNGE ONE (SSO)             │
└─────────────────────────────────────────────────┘
```
